# H Archives

Фіктивний атмосферний сайт архіву організації «H».

## Що всередині
- Гостьовий режим і режим організатора
- Пошук по архіву
- Генератор досьє
- Шум, скан-лінії, редакція та печатки
- Локальне завантаження файлів для прев’ю

## Демо-логін
Логін та пароль організатора зберігаються у вигляді SHA-256 хешів у `script.js`.
Зверніться до адміністратора проєкту для отримання облікових даних.

## Запуск
Просто відкрий `index.html` у браузері або залий на GitHub Pages / Vercel.
